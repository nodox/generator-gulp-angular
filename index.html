What is Wave?

Wave is the full-stack answer to creating projects with HapiJS, AngularJS, and MongoDB. The Wave scaffolding tool will allow you to build scalable, and robust applications in a short amount of time.

Production Ready

Wave was designed with production readiness in mind. New applications are configured for AWS. Capistrano integration will allow you to create a deployment pipeline in your application.

Develop Faster

The Wave development environment is completely automated with GulpJS and Nodemon. Spend more time coding and less time writing configurations.

Get Involved!

Wave thrives on the contributions made by the community. Open an issues whenever a problem arises to keep us informed. With your help we can keep this project alive and current.
